Things to know before you start

Apps use the shadow DOM to create reusable components that are not affected by js and css running on the page. It makes it simpler for customers' devs to reuse these components across their app without worrying about conflicts.

As such, this functionality presents a few unique but straightforward challenges for tagging and guide targeting in pendo.
